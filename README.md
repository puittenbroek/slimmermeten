# The Slimmermeten Project #

## About ##

Django Project to read and visualize power/gas usage from a (Dutch) 'smart meter' via a P1 exit.


## Prerequisites ##

- Python 2.6 or 2.7
- pip
- virtualenv (virtualenvwrapper is recommended for use during development)
